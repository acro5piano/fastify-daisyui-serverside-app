# fastify-daisyui-serverside-app

An example of traditional CRUD app without no Frontend framework. Just HTML.

# Stacks

- Fastify
- react-dom/server (as a template)
- Tailwind CSS
- Daisy UI

# Getting Started

```
pnpm install
pnpm dev
```
